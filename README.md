# 点击浮动特效

一个用于 `Tampermonkey / Violentmonkey / ScriptCat` 的网页点击特效脚本。

启用后，在网页任意位置点击鼠标，即可随机生成 `emoji`、`爱心`、`闪光符号`、`颜文字` 等内容，并以漂浮、旋转、淡出的形式显示在点击位置附近，适合用来增强网页交互氛围。

## 功能特点

- 支持点击网页任意位置触发浮动特效
- 内置多种素材分类：爱心系、闪光系、可爱 Emoji、颜文字
- 支持按权重随机抽取不同分类内容
- 支持一次点击随机生成 `1~3` 个特效，效果更丰富
- 每次特效都会随机变化：轨迹、旋转角度、缩放、透明度、动画时长、缓动曲线
- 使用柔和配色，整体风格偏轻盈、可爱、不过分刺眼
- 带有重复注入保护，避免脚本多次初始化导致重复绑定事件
- 自动清理特效元素，并限制页面最大活跃元素数量，长时间使用也更稳定
- 顶部提供中文配置区，方便按需修改参数

## 效果说明

脚本运行后，点击网页时会在点击位置附近随机出现：

- 爱心类符号，如 `❤`、`💖`、`💘`
- 闪光与装饰类符号，如 `✨`、`🌟`、`🌸`
- 可爱 Emoji，如 `🥰`、`😊`、`🎉`
- 颜文字，如 `(｡･ω･｡)`、`(≧▽≦)`、`(づ｡◕‿‿◕｡)づ`

这些内容会以随机方式向上漂浮，并伴随轻微散开、旋转、缩放与淡出效果。

## 安装方法

### 方式一：本地导入

1. 先安装浏览器脚本管理器：
   - [Tampermonkey](https://www.tampermonkey.net/)
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [ScriptCat](https://scriptcat.org/)
2. 打开脚本管理器的“新建脚本”页面
3. 将 [`点击浮动特效v1.0.0.js`](./点击浮动特效v1.0.0.js) 的内容全部复制进去
4. 保存并启用脚本
5. 打开任意网页测试点击效果

### 方式二：GitHub 发布后安装

如果你把这个仓库发布到 GitHub，可以：

1. 打开脚本原文件
2. 点击 `Raw`
3. 使用脚本管理器直接安装

## 使用方法

1. 确保脚本已在脚本管理器中启用
2. 打开任意网页
3. 在页面空白处或任意可点击区域点击鼠标
4. 即可看到随机浮动的 `emoji / 颜表情 / 爱心 / 闪光` 特效

默认情况下，脚本会匹配所有网页：

```js
// @match        *://*/*
```

如果你只想在指定网站使用，可以自行修改脚本头部的 `@match` 规则。

## 配置说明

脚本顶部提供了中文配置区，常见可调项如下：

```js
const 配置 = Object.freeze({
    最大活跃元素数: 32,
    大小范围: [18, 30],
    时长范围毫秒: [1350, 2350],
    横向散开范围: [-14, 14],
    纵向散开范围: [-10, 10],
    漂移横向抖动范围: [-10, 10],
    漂移纵向抖动范围: [-14, 14],
    初始旋转范围: [-8, 8],
    结束旋转抖动范围: [-8, 8],
    消失透明度范围: [0, 0.12],
    入场缩放范围: [0.88, 1.02],
    离场缩放抖动范围: [-0.03, 0.08],
    连发配置: [
        { 数量: 1, 权重: 56 },
        { 数量: 2, 权重: 32 },
        { 数量: 3, 权重: 12 }
    ],
    启用分类: ['爱心系', '闪光系', '可爱Emoji', '颜文字'],
    分类权重: {
        爱心系: 24,
        闪光系: 20,
        可爱Emoji: 26,
        颜文字: 30
    }
});
```

### 主要配置项解释

- `最大活跃元素数`
  控制页面上同时存在多少个特效元素，数值越大越热闹，但页面负担也会略高。

- `大小范围`
  控制特效字体大小的随机范围。

- `时长范围毫秒`
  控制每个特效持续多久。

- `横向散开范围 / 纵向散开范围`
  控制一次点击触发多个特效时的散开距离。

- `漂移横向抖动范围 / 漂移纵向抖动范围`
  控制漂浮轨迹的随机偏移强度。

- `连发配置`
  控制一次点击生成几个特效，以及每种数量出现的概率权重。

- `启用分类`
  控制启用哪些素材分类。

- `分类权重`
  控制各分类被抽中的概率，数值越大越容易出现。

## 自定义建议

### 如果你想让效果更热闹

- 提高 `最大活跃元素数`
- 提高 `连发配置` 中 `2` 和 `3` 的权重
- 增大 `横向散开范围`、`纵向散开范围`

### 如果你想让效果更柔和

- 缩小 `大小范围`
- 缩小 `漂移横向抖动范围`
- 缩小 `结束旋转抖动范围`

### 如果你只想保留颜文字

把：

```js
启用分类: ['爱心系', '闪光系', '可爱Emoji', '颜文字']
```

改成：

```js
启用分类: ['颜文字']
```

## 兼容性

- Chrome / Edge
- Firefox（需安装对应脚本管理器）
- 基于 Chromium 的大多数浏览器

## 注意事项

- 某些高安全策略页面、浏览器内置页面或扩展页可能无法注入脚本
- 如果网页本身拦截了非常特殊的事件，表现可能会略有差异
- 如果你在低性能设备上使用，建议适当降低连发概率和最大活跃元素数
